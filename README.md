# PDF Viewer - Mobile Friendly

Một ứng dụng web để xem file PDF, tối ưu hóa cho thiết bị di động.

## Features

✅ Xem PDF trực tiếp trên trình duyệt
✅ Responsive design - hoạt động tốt trên mobile
✅ Điều hướng trang trước/sau
✅ Hiển thị số trang hiện tại
✅ Tự động scale PDF theo kích thước màn hình
✅ Không cần backend - chỉ là static files

## Cách sử dụng

1. Đưa file PDF vào thư mục gốc (tên là `starry.pdf`)
2. Mở `index.html` trong trình duyệt
3. Xem và điều hướng PDF

## Deploy lên Vercel

1. Push code lên GitHub
2. Connect repo với Vercel
3. Vercel sẽ tự động deploy

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

## Yêu cầu

- Trình duyệt hiện đại hỗ trợ HTML5 Canvas
- Không cần cài đặt dependencies
